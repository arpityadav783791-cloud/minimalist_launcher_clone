# Minimalist Launcher

A distraction-free Android launcher built with Flutter and Kotlin.

## ✨ Features

- 🔍 Instant app search
- ⭐ Favorites
- 🙈 Hide apps
- 🚀 Native app launching
- 🎯 Minimal text-first UI

## 🚧 Upcoming Features

- Mindful Delay
- Focus Mode
- App Usage Statistics
- App Limits
- Themes
- Backup & Restore

## 🛠️ Tech Stack

- Flutter
- Dart
- Kotlin
- GetX
- Method Channels
- SharedPreferences
